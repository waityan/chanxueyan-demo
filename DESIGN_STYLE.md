# 产学研小程序 - 设计风格指南

## 一、设计品牌定位

### 核心关键词
**专业 · 现代 · 友好 · 移动优先**

- **专业感**：适合作学企合作、职业教育场景的稳重气质
- **现代感**：圆润、简洁、去除冗余装饰
- **友好感**：大圆角、柔和阴影、低对比度避免生硬
- **移动优先**：以手机端体验为核心，适配单手持握

---

## 二、视觉语言特征

### 2.1 形态语言：圆润卡片美学

**设计原则**：
- 拒绝尖锐直角，所有容器采用 **大圆角**（2rem/32px）
- 元素之间使用卡片隔离，形成清晰的"卡片森林"
- 层级通过阴影深度而非边框区分

**形态公式**：
```
界面 = 圆形按钮 + 圆角卡片 + 柔和阴影 + 玻璃元素
```

**典型应用**：
- 主卡片：32px 圆角
- 列表项：24px 圆角
- 头像外框：40px 圆角 (2.5rem)
- 按钮：24px 圆角
- 标签：12px 圆角

### 2.2 色彩语言：蓝色系商务风

**品牌基因**：
```
主色：#2563eb（科技蓝）
辅助：#10b981（成功绿）
强调：#f59e0b（活力橙）
```

**配色逻辑**：
- **主导航/核心按钮** → 品牌蓝
- **价格/高亮** → 活力橙（视觉抓取）
- **免费/成功** → 成功绿（语义化）
- **文字层级** → 深灰到浅灰的自然过渡
- **背景** → 极浅灰蓝 (#f8fafc) 减少视觉疲劳

**色彩情绪**：
- 蓝色占比 60-70%（专业、信任）
- 中性色占比 25-30%（内容承载）
- 强调色占比 5-10%（关键引导）

### 2.3 空间语言：松散呼吸感

**间距哲学**：
- **疏松有致**：卡片间距 20-40px，拒绝拥挤
- **黄金比例**：标题与正文间距 ≈ 标题字号的 2 倍
- **边距统一**：页面内容左右 24px 安全边距

**视觉节奏**：
```
Header (64px)
    ↓ 24px 间距
Section 1 标题区
    ↓ 32px 间距
卡片组（横向滚动或多列）
    ↓ 40px 间距
Section 2 标题区
...
Bottom Nav (60px)
```

### 2.4 材质语言：玻璃与纸的融合

**混合质感**：
- **玻璃拟态**：底部导航栏使用半透明毛玻璃效果（backdrop-filter blur）
  ```css
  background: rgba(255, 255, 255, 0.9);
  backdrop-filter: blur(16px);
  ```
- **纸张质感**：卡片使用纯白背景 + 极淡阴影，模拟浮在页面上的纸张
- **光影**：主要按钮添加品牌色辉光（shadow-blue-100）

## 三、排版风格

### 3.1 字体气质：刚柔并济

**字体选择**：
- 苹方 / 微软雅黑（无衬线，中性和易读）
- 不使用衬线体（避免传统感）

**字重对比**：
- 标题：**Weight 800 (Black)** - 极其粗壮，现代感强
- 正文：**Weight 400 (Regular)** - 简洁舒适
- 标签：**Weight 600 (SemiBold)** - 适中清晰

**字号阶梯**：
```
H1: 28px (Black)
H2: 24px (Black)
H3: 20px (Bold)
H4: 18px (Bold)
Body: 14px (Regular)
Caption: 12px (Regular)
Label: 10px (SemiBold)
```

**排版特色**：
- 无衬线系统字体 → 现代、干净
- 黑色与深灰的对比 → 专业但不刺眼
- 紧凑行距（1.4-1.6）→ 信息密度适中

## 四、交互风格

### 4.1 动效设计：即刻反馈

**核心原则**：**快、准、轻**

- **按压反馈**：scale 从 1 → 0.96，仅 0.1s，模拟真实按钮弹性
- **页面切换**：0.25s 淡入，流畅但不拖沓
- **过渡曲线**：ease-out，快速启动、缓慢结束

**禁用动效**：
- 不使用弹簧动画（ease-out 足够了）
- 不使用复杂视差（保持性能）

### 4.2 状态反馈：清晰直接

- **Toast 提示**：居中、深半透明背景、白色文字、1600ms 自动消失
- **悬浮状态**：轻微上移（transform translateY -2px）+ 阴影加深
- **加载状态**：骨架屏或占位动画（当前版本未实现）

### 4.3 导航逻辑：底部优先

**移动端范式**：
- 三级 Tab 切换（首页/职位/我的）
- 详情页悬浮返回按钮（固定定位 + 玻璃态）
- 子页面顶部返回栏（清晰路径）

## 五、组件设计风格

### 5.1 卡片组件：圆润浮岛

**视觉特征**：
- 大圆角（24-32px）营造柔和感
- 白色背景 + 极淡阴影（card-shadow）
- 可选轻边框（border-slate-50）增加边界感

**变体**：
- **活动卡片**：大图 + 标签 + 多公司徽章
- **课程卡片**：左小图 + 右信息（紧凑布局）
- **职位卡片**：全宽 + 薪资右对齐

### 5.2 按钮组件：品牌发光

**主按钮**（Primary）：
- 品牌蓝背景
- 白色文字 + Bold 字重
- 深阴影 + 品牌色辉光（shadow-blue-100）
- 按压缩小反馈（scale 0.95）

**次要按钮**（Secondary）：
- 白底 + 灰色边框
- 文字使用中性色

### 5.3 标签组件：胶囊形态

**设计统一**：
- 全圆角或大圆角（rounded-md）
- 极小字号（10-11px）但高对比度
- 蓝底白字 或 白底蓝字

**语义映射**：
- 热门/品牌 → 蓝底白字
- 技能关键词 → 白底深灰 + 大写
- 价格强调 → 橙色文字（非背景）

### 5.4 步骤组件：虚线连接

**参访路线可视化**：
- 垂直虚线引导视觉流
- 圆形数字徽章（蓝底白字 + 蓝辉光）
- 每个步骤用浅灰卡片框出信息

## 六、设计原则总结

### 6.1 核心设计哲学

> **"专业而不刻板，现代而不浮夸"**

- **减法设计**：去除不必要的装饰线，依赖间距和负空间
- **材质统一**：玻璃 + 纸张 + 阴影的混合质感
- **色彩克制**：主色仅用于核心操作区域，避免滥用
- **圆润边界**：所有元素都"软"下来，降低攻击性

### 6.2 视觉关键词

| 维度 | 关键词 | 具体表现 |
|------|--------|---------|
| 形状 | 圆润 | 大圆角、椭圆、圆形按钮 |
| 质感 | 轻盈 | 玻璃态、淡阴影、留白 |
| 色彩 | 柔和 | 主色饱和度适中、避免纯黑 |
| 空间 | 通透 | 宽松间距、层次分明 |
| 字体 | 简洁 | 系统字体、黑白对比 |
| 动效 | 灵巧 | 快速响应、短时过渡 |

### 6.3 设计禁忌

- ❌ 纯黑文字（使用 #0f172a 或 #1e293b）
- ❌ 尖锐直角（至少 12px 圆角）
- ❌ 过度饱和配色（饱和度控制在 60-80%）
- ❌ 密集信息堆砌（保持充足的呼吸空间）
- ❌ 复杂的动画或视差效果

## 七、设计风格公式（一句话总结）

> **"以蓝色系商务风为基调，通过大圆角卡片、玻璃拟态和柔和阴影构建移动优先的现代界面，以快速按压反馈和淡入动画营造轻盈交互体验。"**

## 八、应用场景建议

### 适用产品类型
- ✅ 教育/培训类 App
- ✅ 企业服务 B2B 平台
- ✅ 垂直社区/内容平台
- ✅ 工具类应用（商务场景）

### 不适用场景
- ❌ 游戏化产品（过于严肃）
- ❌ 短视频/娱乐 App（过于克制）
- ❌ 极客工具（缺乏个性）
- ❌ 奢侈品电商（不够奢华）

---

## 九、设计验证清单

快速评估一个新组件/页面是否符合设计风格：

- [ ] **形态**：是否所有矩形都有圆角（至少 12px）？
- [ ] **色彩**：是否使用品牌蓝作为主色？是否避免了纯黑纯白？
- [ ] **空间**：元素间距是否 ≥ 12px？是否有足够留白？
- [ ] **字体**：是否使用系统无衬线字体？字重是否清晰？
- [ ] **质感**：是否应用了阴影或玻璃效果？
- [ ] **动效**：交互是否有 0.1-0.25s 的反馈？
- [ ] **整体**：是否传递出"专业、现代、友好"的感觉？

---

## 十、风格演变方向

如果未来需要调整设计风格，可以考虑：

1. **更深邃感**：增加深色模式，使用深蓝/黑色背景
2. **更年轻化**：增加橙色/紫色点缀，使用更大字号对比
3. **更商务化**：减小圆角（12-16px），增加线条装饰
4. **更原生感**：使用 iOS/Android 原生组件风格
